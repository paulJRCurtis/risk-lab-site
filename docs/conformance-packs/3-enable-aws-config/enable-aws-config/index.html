Enable AWS Config

We will enable AWS Config to begin change tracking of your resources.

  1. Select the Services dropdown on the menu bar at the top left. Select Config under Management & Governance. If this is your first time using AWS Config, select Get started. If youâ€™ve already used AWS Config, select Settings.

  1. In the Settings page, under General Settings - Resource types to record, select Record all resources supported in this region checkbox and check the checkbox for Include global resources.

  1. Under Amazon S3 bucket, select Create a bucket.

  2. Optional - Under Amazon SNS topic, check the box for Stream configuration changes and notifications to an Amazon SNS topic, and then select Create a topic, and give the SNS Topic a name. Something like “Conformance-Alerts”, or what ever you like.
    Click Save.

  1. Under AWS Config role, choose Create AWS Config service-linked role.

  2. Click Next

  3. Click Skip to go to Review page and Click Confirm. You can now progress to the next step.
